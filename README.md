# TG_Bot_LM_Studio
 
Telegram бот с поддержкой контекста диалога, использующий локальную языковую модель через LM Studio.

## Функциональность бота
* команды: 
   - /start - начать работу с ботом
   - /model - показать используемую языковую модель
   - /clear - очистить историю диалога
   - /context - показать текущий контекст
* запросы пользователя пересылаются LLM, запущеной на этом компьютере, и потом ответ пересылаются пользователю

## Особенности бота
- бот запоминает историю диалога для каждого пользователя
- независимые диалоги для разных пользователей
- использование языковых моделей через LM Studio

## Установка и запуск

1. Клонируйте репозиторий:
git clone https://github.com/ВАШ_USERNAME/TG_Bot_LM_Studio.git
cd TG_Bot_LM_Studio

2. Установите зависимости:
pip install -r requirements.txt

3. Запуск
python main.py
